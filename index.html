Documentation for Setting Up extention

To get started with creating an extension on Fynd, set up your development environment by reviewing Fynd's API documentation and integration guidelines. These resources will help you understand essential endpoints, authentication, and data flows within the platform. Then, create a structured project that organizes code effectively, ensuring smooth functionality and user experience. Finally, follow best practices for testing and debugging to ensure your extension integrates seamlessly with Fynd.

Online Web Setup / Becoming Partner

Step 1: Register Yourself

To get started, Register yourself as a fynd partner first

https://partner.fynd.com

Step 2: Verify Yourself

Verify your phone number & email by providing OTPs

Step 3: Create a Main Partner Company

Create a partner company with all details asked like (name, mobile no, etc)

Step 4: Create a Development Company

Create a Development company as it will be required for pulling initial dummy data for your extension

PC Setup

To get started, you'll need the following:

Git for the starter code Node.js 18+ for FDK CLI Partner account ( refer above to create a partner account ) Development company ( refer above to create a development company )

Step 1: Install FDK CLI

The FDK CLI simplifies theme and extension development using commands for creating, testing, and deploying projects. For more details, refer to the GitHub documentations.

Run the following command in your terminal to install FDK CLI:

npm install -g @gofynd/fdk-cli

Step 2: Login

Run the following command to log into your Fynd Partner account:

fdk login

To complete login go on the link provided and select a company and done

Step 3: Creating a new extension

To create a new extension run the following command

fdk extension init

Fill up all the details asked

Enter Name

Select Type

Select the preffered tech stack ( the code for your extension will use this stack )

Step 4: Install Dependencies

Install necessary dependencies for the project. In this example, we are using Node.js.

npm install express

Step 4: Populate Test Data

Switch to your project directory

cd 'extension-name

Run the following command to get test data

fdk populate

Select the development compnay you want to use

Step 5: Preview Your Extension

Run the following command:

fdk extension preview

Congrats!! you have made your first extension
